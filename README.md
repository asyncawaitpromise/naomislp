# Naomi's Speech Therapy Practice

A professional website for Naomi's private speech language pathology practice, specializing in cleft palate speech therapy.

## About

This website serves as the homepage for Naomi's speech therapy practice, providing information about her specialized services in cleft palate speech therapy. The site features a welcoming design and comprehensive information about treatment approaches and contact details.

## Features

- Professional homepage showcasing speech therapy services
- Specialized focus on cleft palate therapy
- Responsive design with Tailwind CSS and DaisyUI
- Contact information and consultation booking
- Clean, accessible interface for patients and families

## Technology Stack

- **Frontend**: React 18 with Vite
- **Routing**: React Router DOM
- **Styling**: Tailwind CSS + DaisyUI
- **Icons**: React Feather
- **Build Tool**: Vite
- **Package Manager**: pnpm

## Getting Started

To run this project locally:

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Or start with host access on port 80
pnpm host
```

## Available Scripts

- `pnpm dev` - Start development server
- `pnpm host` - Start development server with host access on port 80
- `pnpm build` - Build for production
- `pnpm lint` - Run ESLint
- `pnpm preview` - Preview production build
